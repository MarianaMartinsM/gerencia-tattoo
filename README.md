# Gerencia Tattoo

Sistema de gerenciamento de tatuagens desenvolvido para a disciplina de Projeto de Software.

## Tecnologias utilizadas
- React
- Node.js
- Express
- MySQL

## Funcionalidade AC1
Implementação do banco de dados e criação da API para cadastro e listagem de tatuagens.

## Estrutura do projeto

front-end → interface do sistema  
back-end → API e conexão com banco de dados
